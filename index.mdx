    Get up and running with the Sentry Go SDK in just a few steps.
  </p>
  
  <Steps>
    <Step title="Install the package">
      Add the Sentry Go SDK to your project using Go modules:
      
      ```bash
      go get github.com/altipla-consulting/sentry
      ```
    </Step>
    
    <Step title="Initialize the client">
      Create a new Sentry client with your DSN:
      
      ```go
      import "github.com/altipla-consulting/sentry"

      func main() {
          client := sentry.NewClient("your-sentry-dsn")
          defer client.Flush(5 * time.Second)
      }
      ```
      
      <Note>
        If you pass an empty DSN, the client will be nil but can still be used safely - it simply won't report anything.
      </Note>
    </Step>
    
    <Step title="Report your first error">
      Use the client to report errors with context:
      
      ```go
      ctx := sentry.WithContext(context.Background())
      
      if err := someOperation(); err != nil {
          client.Report(ctx, err)
      }
      ```
    </Step>
  </Steps>
